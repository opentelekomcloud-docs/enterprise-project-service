How Do I Obtain an Enterprise Project ID?
=========================================

-  By calling an API

   Cloud services can be authorized by IAM to call the API that is used to query a list of enterprise projects. The returned project list contains the ID of each enterprise project. For details about the APIs, see *Enterprise Project Management Service API Reference*.

-  From the enterprise project details page.

   The procedure is as follows:

   #. Sign in to the management console, click the account name in the upper right corner, and choose **Enterprise Management** from the drop-down list.

      The **Enterprise Project Management** page is displayed.

   #. Click the name of the target enterprise project to view its ID on the displayed details page.
