Viewing Migration Events
========================

Scenarios
---------

You can view details of migration events, including time, operator, operation, resource ID, source enterprise project, and destination enterprise project of each event.

Limitations and Constraints
---------------------------

Events in the last seven days can be queried.

To view migration events, you must have the administrator permission.

Procedure
---------

#. On the management console, click **username** in the upper right corner and choose **Enterprise Management** from the drop-down list.

   The **Enterprise Project Management** page is displayed.

#. In the upper left of the enterprise project list, click **View Migration Event**.

   The **Migration Events** page is displayed, listing the resource adding and removal events.
