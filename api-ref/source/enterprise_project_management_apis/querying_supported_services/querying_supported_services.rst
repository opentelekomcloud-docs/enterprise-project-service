Querying Supported Services
===========================

Function
--------

You can use this API to query services supported by EPS.

URI
---

GET /v1.0/enterprise-projects/providers

.. table:: **Table 1** Query parameters

   +-----------+-----------+---------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | Parameter | Mandatory | Type    | Description                                                                                                                                                                           |
   +===========+===========+=========+=======================================================================================================================================================================================+
   | limit     | No        | Integer | Default value: 10, upper limit: 200, lower limit: 1.                                                                                                                                  |
   +-----------+-----------+---------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | offset    | No        | Integer | Specifies the index position. The query starts from the next data record specified by offset. The value must be a number and cannot be a negative number. The default value is **0**. |
   +-----------+-----------+---------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | provider  | No        | String  | Cloud service name                                                                                                                                                                    |
   +-----------+-----------+---------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

Request
-------

.. table:: **Table 2** Parameters in the request header

   +--------------+-----------+--------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | Parameter    | Mandatory | Type   | Description                                                                                                                                                                                                 |
   +==============+===========+========+=============================================================================================================================================================================================================+
   | X-Auth-Token | Yes       | String | Specifies the user token. EPS is a global service. When calling the IAM API to obtain a user token, set **scope** to **domain**. The value of **X-Subject-Token** in the response header is the user token. |
   +--------------+-----------+--------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

Response
--------

**Status code: 200**

.. table:: **Table 3** Parameters in the response body

   +-------------+--------------------------------------------------------------------------------------------------------------------------+-----------------------------------------------+
   | Name        | Type                                                                                                                     | Description                                   |
   +=============+==========================================================================================================================+===============================================+
   | providers   | Array of :ref:`ProviderResponseBody <listproviders__en-us_topic_0000001430428421_response_providerresponsebody>` objects | Specifies cloud services.                     |
   +-------------+--------------------------------------------------------------------------------------------------------------------------+-----------------------------------------------+
   | total_count | Integer                                                                                                                  | Specifies the total cloud services supported. |
   +-------------+--------------------------------------------------------------------------------------------------------------------------+-----------------------------------------------+

.. _listproviders__en-us_topic_0000001430428421_response_providerresponsebody:

.. table:: **Table 4** ProviderResponseBody

   +----------------------------+------------------------------------------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------------+

   +----------------------------+------------------------------------------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------------+
   | provider_i18n_display_name | String                                                                                                           | Specifies the display name of the cloud service. You can set the language by setting the **locale** parameter. |
   +----------------------------+------------------------------------------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------------+
   | resource_types             | Array of :ref:`ResourceTypeBody <listproviders__en-us_topic_0000001430428421_response_resourcetypebody>` objects | Specifies the resource type.                                                                                   |

.. table:: **Table 5** ResourceTypeBody

Example Request
---------------

Querying supported services

.. code-block:: text

   GET https://{Endpoint}/v1.0/enterprise-projects/providers

Example Response
----------------

**Status code: 200**

.. code-block::

   {
     "providers" : [ {
       "provider" : "evs",
       "provider_i18n_display_name" : "Elastic Volume Service",
       "resource_types" : {
         "resource_type_i18n_display_name" : "volume",
         "global" : false,
         "resource_type" : "disk",
         "regions" : [ "regionid1" ]
       }
     } ],
     "total_count" : 1
   }
